# Structured Plan Execution

## Principles
1. **Incrementalism**: Small, verifiable steps.
2. **Validation**: Every step must have a verification step (e.g., run a test, check a log).
3. **State Management**: Keep track of what's done, what's pending, and any new learnings.
4. **Backtracking**: If a step fails, diagnose why and adjust the plan.

## Execution Cycle
- **Plan**: Briefly state the goal of the next step.
- **Act**: Execute the change.
- **Validate**: Verify the change worked.
